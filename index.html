/// 결과 보고서닌텐도의 삼인칭 슈팅 비디오 게임 시리즈 '스플래툰'의 스토리를 소개하는 사이트에 새로운 레이아웃을 적용해 보았습니다. 메뉴에 마우스 오버할 경우 백그라운드에 동영상이 재생됩니다. 메뉴 버튼을 클릭할 경우 각 메뉴에 대항되는 내용이 이미지로 올라옵니다.

/// 결과 보고서모나미의 프리미엄 펜 라인을 소개하는 페이지를 제작하였습니다. 좌우 슬라이드로 이동하며, 이미지에 마우스 오버를 할 경우 좌측의 대표 이미지와 제품 소개 글이 변경되는 기능을 가지고 있습니다.

/// 결과 보고서자취생, 1인 가구 등을 대상으로 한 온라인 쇼핑몰 원룸만들기 사이트 내부의 3가지 카테고리를 골라 대표 상품들을 배치하고 소개하는 페이지를 제작하였습니다. 좌측 상단의 버튼을 클릭할 경우 메뉴가 나타나며, 각각의 메뉴를 클릭하면 화면전환 효과와 함께 페이지를 이동합니다.

/// 결과 보고서제주맥주에서 판매하고 있는 6종류의 맥주를 간단하게 소개하는 페이지를 제작하였습니다. 좌우에 배치되어 있는 맥주 이미지를 클릭하면 중앙의 큐브가 회전하며 해당 맥주의 소개 글로 변화합니다.

/// 결과 보고서에스쁘아의 공식 홈페이지 소스를 이용하여 간단하게 리뉴얼해보았습니다. 그리드 레이아웃과 fixed position 등이 사용되었습니다.

/// 결과 보고서러쉬의 공식 홈페이지 소스를 이용하여 버튼을 클릭하면 이동하는 슬라이드 홈페이지를 제작하였습니다. 버튼의 종류에 따라 좌측, 우측으로 이동하며 끝까지 이동했을 경우 처음으로 돌아갑니다. 좌측 메뉴로도 이동이 가능합니다.

/// 결과 보고서노플라스틱선데이의 홈페이지를 제작하였습니다. 기존의 홈페이지에 메뉴를 클릭하면 해당 섹션으로 부드럽게 이동하는 기능 등을 추가하였습니다.

/// 결과 보고서넥슨 PC방 사이트의 레이아웃을 이용하여 빽다방 홈페이지를 제작하였습니다. 각각의 메뉴에는 마우스 오버할 경우 전환 효과과 적용되어 있습니다.

/// 결과 보고서롯데 칠성의 홈페이지를 일부 클론 코딩하였습니다. 음료의 종류를 클릭할 경우 위로 올라오는 애니메이션과 함께 하단의 바가 움직입니다. 백그라운드에는 동영상이 적용되어 있습니다.

/// 결과 보고서ifKaKao의 2022년 버젼 페이지를 클론 코딩하였습니다.

/// 결과 보고서뮤고박스 홈페이지의 특정 섹션을 클론 코딩하였습니다. 각 캐릭터에 따라 특정 애니메이션이 적용되어 있습니다.

/// 결과 보고서도미노 피자 홈페이지의 소스를 이용하여 페이지를 제작하였습니다.

/// 결과 보고서청와대, 국민 품으로 홈페이지 개편 이전 버젼을 클론 코딩하였습니다.

/// 결과 보고서엘카페의 홈페이지 일부 요소를 클론 코딩하였습니다.

/// 결과 보고서우담라이팅의 메인 홈페이지를 클론 코딩하였습니다. 좌측의 메뉴를 클릭할 경우 이미지가 변경됩니다.

/// 결과 보고서메일유업의 요소와 동영상을 이용하여 메뉴가 포함되어 있는 페이지를 제작하였습니다. 마우스 오버할 경우 하위메뉴가 등장합니다. 또한 우측 하단에 마우스 오버할 경우 패밀리 사이트가 애니메이션 효과를 가지고 등장합니다.

/// 결과 보고서팝마트코리아의 대표 피규어 등을 소개하는 페이지를 제작하였습니다. 마우스 오버할 경우 피규어 이미지가 움직이고, 동영상이 등장할 때와 사라질 때 효과가 들어갔습니다. 피규어 이미지를 클릭할 경우 해당 제품의 판매 페이지로 이동합니다.

/// 결과 보고서에이스 침대의 홈페이지의 일부 요소를 클론 코딩하였습니다. Swiper를 커스텀하여 슬라이드를 적용하였고, 클릭할 경우 좌우로 이동하는 탭 메뉴가 있습니다.
